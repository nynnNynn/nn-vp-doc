# 070 爬楼梯
## code
```javascript
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
    let dp = [];
    dp[0] = 1;
    // i - 1  0索引 1级台阶
    dp[1] = 2;
    for(let i = 2; i < n ; i++) {
        dp[i] = dp[i-1] + dp[i-2];
    }
    return dp[n - 1];
};
```
## 总结
1. 好工具题，不光是dp，要是太久不写了啥都可以拿这个刷下手感
2. 没啥特殊的 i-1 + i-2就是了
